# Bundled UI fonts

Naza One bundles these open fonts locally so the UI keeps its typography without
runtime network font fetching:

- Inter — main UI font for cleaner desktop readability
- Space Grotesk — bundled accent/display font
- JetBrains Mono — telemetry/status/code-style font

These font families are sourced from Google Fonts and include their OFL license
files in this folder.
